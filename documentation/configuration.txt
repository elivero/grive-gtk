CONFIGURATION
-------------
	The program reads and handles a file located in your '/etc' directory.
	This file contains some basic configuration rules, which are, or should
	at least be self-explaining or made clear by the comments.
	
	Issue the following command to view and edit the file:
		sudo nano /etc/grive-gtk.conf
			
			or
		
		sudo vi /etc/grive-gtk.conf
